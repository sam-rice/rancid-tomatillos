# Rancid Tomatillos

### [Deployed Site](https://rancid-tomatillos-project.netlify.app/)

### Abstract

This project is a multi-page React web application designed to emulate a movie review aggregator website (Ã  la Rotten Tomatoes/IMDb). The application allows a user to browse and rate movies, watch trailers, and view other miscellaneous film stats. All data is served up via a [Node.js/Express backend server](https://github.com/sam-rice/rancid-tomatillos-api). The project also includes a Cypress acceptance test suite.
